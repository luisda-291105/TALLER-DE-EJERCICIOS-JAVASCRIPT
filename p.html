¿Que es un array?

Un array (también llamado arreglo) es una estructura de datos que permite almacenar varios valores en una sola variable. Cada valor dentro del array se guarda en una posición llamada índice, y estos índices comienzan desde el número 0.

Por ejemplo, si quieres guardar varios nombres sin usar muchas variables, puedes hacerlo así en JavaScript: 


    let nombres = ["Ana", "Luis", "María", "Carlos"];

En este caso:

  • nombres[0] contiene "Ana"
  • nombres[1] contiene "Luis"
  • nombres[2] contiene "María"
  • nombres[3] contiene "Carlos"

Los arrays son muy útiles cuando necesitas trabajar con grupos de datos, como listas de productos, números, usuarios o cualquier colección de elementos que tengan algo en común.

¿Que es un objeto?

Un objeto es una estructura de datos que permite almacenar información en forma de pares clave–valor. Esto significa que cada dato dentro del objeto tiene un nombre (la clave) y un valor asociado.

Los objetos son muy útiles para representar cosas del mundo real, como una persona, un producto o un auto, ya que pueden tener diferentes características (propiedades) y comportamientos (métodos).

Por ejemplo, en JavaScript puedes crear un objeto así:


let persona = {
    nombre: "Luis",
    edad: 19,
    ciudad: "Medellín",
    saludar: function() {
        return "Hola, mi nombre es " + this.nombre;
    }
};

En este ejemplo:

  • nombre, edad y ciudad son propiedades.
  • saludar() es un método (una función dentro del objeto).

Puedes acceder a los valores de un objeto usando el punto (.), por ejemplo: persona.nombre devolverá "Luis".

¿Cual es la diferencia?

Aunque los arrays y los objetos son estructuras de datos en JavaScript que permiten almacenar información, se usan para propósitos diferentes y tienen formas distintas de organizar los datos.

Característica Array Objeto
Estructura Lista ordenada de valores con índices numéricos. Pares clave–valor con nombres personalizados.
Acceso frutas[0] persona.nombre
Ejemplo let frutas = ["Manzana", "Banana", "Cereza"]; let persona = { nombre: "Luis", edad: 19 };
Orden Los elementos tienen posiciones (0, 1, 2...) El orden no importa; cada propiedad tiene su nombre.
Uso común Listas o colecciones de datos similares. Representar entidades con atributos y métodos.

En resumen, usa un array para listas ordenadas y un objeto para describir cosas con varias propiedades.
